# 💪🔥 DOOPALIZE 🔥💪

## 🦾 사원증 사진에 강철 같은 업무 방어력을 주입한다

**Doopalize**는 순해 보이는 프로필 사진을  
**더 단단하게 😤 더 강하게 🧱 더 문의하기 어렵게 ⚡** 바꿔주는  
브라우저 기반 초강력 인상 보정기다.

사진을 넣는다.  
슬라이더를 올린다.  
눈빛이 바뀐다.  
분위기가 눌린다.  
문의가 망설여진다.  

## 🚨 핵심 기능 🚨

- 🧨 **두팔 강도 조절**: 순한 인상을 업무 방어형 인상으로 압축
- 🪨 **윤곽 보정**: 흐릿한 분위기를 단단한 존재감으로 강화
- 🕶️ **곽두팔 프리셋**: 이름만 들어도 메신저 창을 닫게 만드는 모드
- 🏢 **사원증 프리셋**: 자연스럽지만 쉽게 건드리기 어려운 회사형 인상
- 👔 **임원실 프리셋**: 말 걸기 전에 일정부터 확인하게 되는 묵직함
- 🛡️ **원본 얼굴선 보존**: 너는 너인데 더 강한 너
- 🌑 **배경 존재감 낮추기**: 얼굴의 압박감만 앞으로 전진
- 💾 **결과 저장**: 완성된 두팔 이미지를 PNG로 저장

## ⚔️ 사용법 ⚔️

1. `index.html`을 연다.
2. 사진을 넣는다.
3. 프리셋을 고른다.
4. `두팔 강도`를 올린다.
5. `윤곽 보정`을 올린다.
6. 결과를 저장한다.
7. 프로필 사진을 교체한다.
8. 세상이 조용해진다.
